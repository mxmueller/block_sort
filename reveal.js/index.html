Mergesort mit Min-Heap

Beispiel

Gegebenes Array

[1, 7, 8, 2, 3, 5, 4, 6]

Blockgröße von 3


			  Block 1: [1, 7, 8]
			  Block 2: [2, 3, 5]
			  Block 3: [4, 6]

Sortieren der Blöcke


			  Block 1: [1, 7, 8]
			  Block 2: [2, 3, 5]
			  Block 3: [4, 6]

Merging der sortierten Blöcke


			  Min-Heap: [1, 2, 4]
			  
			  [1]
			  
			  Min-Heap: [2, 7, 4]
			  
			  [1, 2]
			  
			  Min-Heap: [4, 7, 5]
			  
			  [1, 2, 4]
			  
			  Min-Heap: [5, 7, 8]
			  
			  [1, 2, 4, 5]
			  
			  Min-Heap: [6, 7, 8]
			  
			  [1, 2, 4, 5, 6]
			  
			  Min-Heap: [7, 8]
			  
			  [1, 2, 4, 5, 6, 7]
			  
			  Min-Heap: [8]
			  
			  [1, 2, 4, 5, 6, 7, 8]

Ausgabearray

[1, 2, 4, 5, 6, 7, 8]

Fertig

Komplexität

Im Beispiel mit dem Array [1, 7, 8, 2, 3, 5, 4, 6] ist die Blockgröße 3.

Die Zeitkomplexität beträgt daher
O(n log 3) = O(n log 3).
